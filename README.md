# YXX · WordPress Plugin Developer

Personal portfolio and project showcase. Static site hosted on GitHub Pages.

## Overview

Independent WordPress plugin developer building for the overseas market. 
Each plugin has a capable free version on WordPress.org and a Pro version 
with advanced features, distributed via GitHub Releases with Creem license 
key verification.
